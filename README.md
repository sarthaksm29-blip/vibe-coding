# ITM Makar Sankranti Celebration Website

A responsive, single-page website created to celebrate Makar Sankranti for ITM University. The site features a professional light theme, cultural insights, and interactive animations.

## Features

- **Responsive Design**: Mobile-friendly layout using CSS Grid.
- **Premium Light Theme**: professional color palette (`#FFFDF6` background, Green/Magenta accents).
- **Interactive Animations**:
  - **Floating Kites**: JavaScript-powered kite generation in the background.
  - **Confetti Celebration**: "Send a Wish" interactivity.
- **Cultural Sections**:
  - **About**: Significance of the festival.
  - **Traditions**: Cards explaining Til-Gul, Patang Baazi, Black Attire, and Food.

## Tech Stack
- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid, Animations)
- **Vanilla JavaScript** (DOM Manipulation, Animations)

## Setup
1. Clone the repository.
2. Open `makar_sankranti/index.html` in your browser.

## Customization
- **Colors**: Edit CSS variables in `:root` inside `style.css`.
- **Text**: Content can be modified directly in `index.html`.
